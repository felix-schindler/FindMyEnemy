# Find my enemy (MWA)

## Run the app

Start frontend, backend and database (on [:80](http://localhost)) with `docker-compose up`

## Services

| Service      | Port      | Directory | Path |
|--------------|-----------|-----------|----------|
| Frontend     | 80:3000   | frontend  | /        |
| Backend      | 80:8000   | backend   | /api/    |
| SurrealDB    | 80:8000   | data      | /db/     |
